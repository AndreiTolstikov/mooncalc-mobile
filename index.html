You understand and agree that the MoonCalc Mobile Web App is provided "AS IS" without warranty of any kind.
The MoonCalc Mobile Web App is used the W3C Geolocation API (GeoIP) technology to determine the position of the user. For correct display of the Moon for the residents of the Northern and Southern hemispheres. And the Equator.
For any other purposes location data of the user are not used.
Note: When downloading or updating data in the Web App by clicking "Refresh" button mobile operator may charge additional fee for data transfer.
