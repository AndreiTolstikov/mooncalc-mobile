User agreement

MoonCalc Mobile Web App for calculating Age of the Moon, Phase of the Moon, and Distance from the Moon to the center of Earth. For residents of the Northern Hemisphere the Southern Hemisphere, and the Equator.
You understand and agree that the MoonCalc Mobile Web App is provided "AS IS" without warranty of any kind.
The MoonCalc Mobile Web App is used the W3C Geolocation API (GeoIP) technology to determine the position of the user. For correct display of the Moon for the residents of the Northern Hemisphere, Southern Hemisphere, and the Equator.
For any other purposes location data of the user are not used.
Note:When updating data in the MoonCalc Web App by reload the current page mobile operator may charge additional fee for data transfer.
If there is no Internet connection, the calculation of the parameters of the Moon is performed only for the Northern Hemisphere.
